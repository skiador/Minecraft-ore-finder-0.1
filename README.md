# Minecraft-ore-finder-0.1

To use this code you will need a mod called "Minescript".
You will need to add the mod and then go to the folder of the minecraft and open/create a folder called minescript, then you will need to insert this code inside a .py file, after you do that you can simply go to the game and send in the chat \(nameofthefile)
Keep in mind that you dont need to add the .py in the chat, just the name is good to go
